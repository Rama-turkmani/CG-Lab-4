⚙️ التقنيات المستخدمة

VBO

Vertex Buffer Object - لتخزين بيانات رؤوس الأشكال (المستطيل والمثلث) في ذاكرة GPU

EBO

Element Buffer Object - لرسم المستطيلات من مثلثين باستخدام الفهارس بدل تكرار الرؤوس

VAO

Vertex Array Object - لتنظيم وحفظ إعدادات سمات الرؤوس (مستطيل VAO + مثلث VAO)

Uniforms

متغيرات موحدة - لتمرير الموقع (uOffset) والحجم (uScale) واللون (uColor) ديناميكياً

🎨 الشيدرز (Shaders)

Vertex Shader - تحديد الموقع والحجم

#version 330 core
layout (location = 0) in vec3 aPos;
uniform vec2 uOffset; // إزاحة الموقع
uniform vec2 uScale; // مقياس الحجم
void main() {
  gl_Position = vec4(aPos.x * uScale.x + uOffset.x,
                     aPos.y * uScale.y + uOffset.y,
                     aPos.z, 1.0);
}

Fragment Shader - تحديد اللون

#version 330 core
out vec4 FragColor;
uniform vec4 uColor; // لون الكائن
void main() {
  FragColor = uColor;
}

✨ التحريك باستخدام Uniforms

  • uOffset (vec2) - يُحدد موقع كل كائن على الشاشة
    مثال: تحريك المجداف يميناً ويساراً بتغيير playerX
  • uScale (vec2) - يُحدد حجم كل كائن
    مثال: المجداف (0.3 × 0.06) vs النجمة (0.1 × 0.1) من نفس VBO
  • uColor (vec4) - يُحدد لون كل كائن (RGBA)
    مثال: ألوان نابضة باستخدام sinf(time) لتأثير التلألؤ
// رسم أي شكل بموقع وحجم ولون مختلف:
glUniform2f(loc("uOffset"), x, y);
glUniform2f(loc("uScale"), w, h);
glUniform4f(loc("uColor"), r, g, b, 1.0);
glBindVertexArray(VAO);
glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, 0);

🔄 حلقة اللعبة (Game Loop)

1. معالجة المدخلات (Input)
←
2. تحديث حالة اللعبة (Update)
←
3. الرسم (Render)
←
4. تبديل Buffers + أحداث
  • Input: أسهم لوحة المفاتيح لتحريك المجداف، ESC للخروج، R لإعادة اللعب
  • Update: تحريك النجمة للأسفل، فحص التصادم، تحديث النقاط والحياة
  • Render: رسم الخلفية ← الأرضية ← النجوم ← المجداف ← مؤشرات الحياة

🧩 التحديات والحلول

التحديات

  • رسم أشكال مختلفة الأحجام بدون مصفوفات تحويل
  • تحريك الكائنات بسلاسة
  • فحص التصادم بين الأشكال
  • إدارة حالات اللعبة المختلفة

الحلول

  • استخدام Uniforms للمقياس (uScale) بدل المصفوفات
  • تحديث الموقع كل إطار + VSync لتثبيت السرعة
  • خوارزمية AABB للتصادم بين المستطيلات
  • متغيرات bool (gameOver) + إعادة تعيين بزر R

🎆 التأثيرات البصرية

  • 🌙 خلفية سماء ليلية: لون أزرق غامق يتغير بلطف مع الوقت باستخدام sinf(time)
  • ✨ نجوم خلفية متلألئة: 8 نجوم ثابتة بتأثير تلألؤ (تغيير سطوع اللون)
  • 💫 نجمة ساقطة نابضة: لون النجمة ينبض باستخدام pulse = 0.85 + 0.15 * sin(t*6)
  • 🎨 مجداف متغير اللون: ألوان المجداف تتغير بسلاسة مع مرور الوقت
  • 🔴 شاشة Game Over: خلفية حمراء + علامة + وامضة في المنتصف
  • ❤️ مؤشرات الحياة: مستطيلات حمراء صغيرة في الزاوية العليا
