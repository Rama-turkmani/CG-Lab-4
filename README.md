# 🌟 Catch the Stars! - التقط النجوم

**مشروع عملي لمقرر الرسوميات الحاسوبية (Computer Graphics)**

لعبة 2D بسيطة مبنية باستخدام OpenGL 3.3 مع GLEW و GLFW.

---

## 🎮 فكرة اللعبة

اللاعب يتحكم بمجداف في أسفل الشاشة لالتقاط النجوم الساقطة من الأعلى. كل نجمة يتم التقاطها تزيد النقاط وتزيد سرعة اللعبة. لديك 3 محاولات قبل انتهاء اللعبة.

## 🕹️ طريقة اللعب

| المفتاح | الوظيفة |
|---------|---------|
| `←` / `A` | تحريك المجداف لليسار |
| `→` / `D` | تحريك المجداف لليمين |
| `R` | إعادة اللعب (بعد Game Over) |
| `ESC` | الخروج من اللعبة |

## ⚙️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **VBO** (Vertex Buffer Object) | تخزين رؤوس المستطيل (4 رؤوس) والمثلث (3 رؤوس) في ذاكرة GPU |
| **EBO** (Element Buffer Object) | رسم المستطيلات من مثلثين باستخدام 6 فهارس بدل 6 رؤوس |
| **VAO** (Vertex Array Object) | تنظيم سمات الرؤوس - `rectVAO` للمستطيلات و `triVAO` للمثلثات |
| **Vertex Shader** | حساب الموقع النهائي: `position = vertex × scale + offset` |
| **Fragment Shader** | تطبيق اللون الديناميكي على كل بكسل |
| **Uniforms** | `uOffset` (الموقع) + `uScale` (الحجم) + `uColor` (اللون) |

## 🎨 العناصر المرئية

- **سماء ليلية** - خلفية زرقاء غامقة تتغير مع الوقت
- **نجوم خلفية** - 8 نجوم ثابتة بتأثير تلألؤ (sinf)
- **مجداف اللاعب** - مستطيل بلون متغير ديناميكياً
- **نجمة ساقطة** - مثلث بلون عشوائي ونبض ضوئي
- **مؤشرات الحياة** - مستطيلات حمراء صغيرة (أعلى اليسار)
- **شاشة Game Over** - خلفية حمراء + علامة + وامضة

## 🛠️ متطلبات التشغيل

- **Visual Studio 2022** (أو أحدث) مع C++ Desktop Development
- **OpenGL 3.3** متوافق (معظم كروت الشاشة الحديثة)
- مكتبات GLEW و GLFW مضمنة في المشروع (`External Libraries/`)

## 🚀 خطوات التشغيل

1. افتح `lect4/project4/project1.slnx` في Visual Studio
2. اختر **Debug | x64** من شريط الأدوات
3. اضغط `Ctrl + F5` لتشغيل المشروع
4. استمتع باللعبة! 🎮

## 📋 ملاحظات تقنية

- الشيدرز مكتوبة inline في الكود (لا ملفات خارجية)
- يُستخدم VSync (`glfwSwapInterval(1)`) لتثبيت سرعة اللعبة
- المسارات في `.vcxproj` تستخدم `$(ProjectDir)` النسبية (لا مسارات مطلقة)
- تم استخدام `GLEW_STATIC` للربط الساكن مع `glew32s.lib`
